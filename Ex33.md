## 33. Liệt kê email user các user có tên(user.full_name) có chứa ký tự "Khi" theo danh sách như output bên dưới. 
```sql
SELECT 
    GROUP_CONCAT(DISTINCT `email`
        SEPARATOR '; ') AS emails_user
FROM
    user
WHERE
    full_name LIKE '%Khi%';
```
