## 19. Lấy 5 blog và 5 news của 1 category bất kỳ
```sql
set @cat = (select id from category where id >= floor(rand()*(select max(id) from category)) group by id limit 1);
(SELECT 
    category_id, title, view
FROM
    blog
WHERE
    category_id = @cat
ORDER BY RAND()
LIMIT 5) UNION (SELECT 
    category_id, title, view
FROM
    news
WHERE
    category_id = @cat
ORDER BY RAND()
LIMIT 5); 
```
