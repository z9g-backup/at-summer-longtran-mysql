## 14. Lấy 3 User comment đầu tiên trong 5 blogs mới nhất
```sql
SELECT 
    *
FROM
    comment
        INNER JOIN
    (SELECT 
        id
    FROM
        blog
    ORDER BY created_at DESC
    LIMIT 5) AS v2 ON comment.target_id = v2.id
GROUP BY (user_id)
ORDER BY created_at ASC
LIMIT 3;
```
