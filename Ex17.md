## 17. Select 10 blog mới nhất được tạo bởi các user active
```sql
SELECT 
    blog.*
FROM
    blog,
    user
WHERE
    blog.user_id = user.id
        AND user.is_active = 1
ORDER BY blog.created_at DESC
LIMIT 10;
```
