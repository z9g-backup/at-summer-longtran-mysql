## 34. Tính điểm cho user có email là minh82@example.com trong bảng comment.
```sql
SELECT 
    SUM(CASE
        WHEN (target_table = 'blog') THEN 2
        ELSE 1
    END)
FROM
    comment
        INNER JOIN
    user ON comment.user_id = user.id
WHERE
    user.email = 'minh82@example.com';
```
