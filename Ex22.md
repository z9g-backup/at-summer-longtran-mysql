## 22. Lấy danh sách user đã comment trong 2 blog mới nhất
```sql
SELECT 
    user.*
FROM
    user,
    comment
        INNER JOIN
    (SELECT 
        id
    FROM
        blog
    ORDER BY created_at DESC
    LIMIT 2) AS v2 ON comment.target_id = v2.id
WHERE
    comment.user_id = user.id
        AND comment.target_table = 'blog';
```
